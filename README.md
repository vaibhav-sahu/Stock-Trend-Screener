# Stock-Trend-Screener
Stock Trend Screener
Stock Trend Screener is a data-driven tool designed to analyze historical market trends and provide short-term price forecasts. By evaluating the last 3 months of price action, the application generates visual insights and predicts the next day's price movement to help traders make informed decisions.
# Key Features
3-Month Trend Analysis: Automatically fetches and processes the last 90 days of historical data.
Next-Day Prediction: Uses to forecast the upcoming market price.
Interactive Visualizations: Generates clean, easy-to-read graphs showing historical trends alongside the predicted value.
